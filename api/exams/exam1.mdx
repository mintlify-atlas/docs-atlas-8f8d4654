## Submit Exam 1 Responses

<RequestExample>
```bash cURL
curl -X POST https://proyecto-ingenieria-software-6ccv.onrender.com/guardar_examen1 \
  -H "Content-Type: application/json" \
  -d '{
    "email": "student@example.com",
    "r1": "answer1",
    "r2": "answer2",
    "r3": "answer3",
    "r4": "answer4",
    "r5": "answer5",
    "r6": "answer6"
  }'
```

```javascript JavaScript
const response = await fetch('https://proyecto-ingenieria-software-6ccv.onrender.com/guardar_examen1', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    email: 'student@example.com',
    r1: 'answer1',
    r2: 'answer2',
    r3: 'answer3',
    r4: 'answer4',
    r5: 'answer5',
    r6: 'answer6'
  })
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

url = "https://proyecto-ingenieria-software-6ccv.onrender.com/guardar_examen1"
payload = {
    "email": "student@example.com",
    "r1": "answer1",
    "r2": "answer2",
    "r3": "answer3",
    "r4": "answer4",
    "r5": "answer5",
    "r6": "answer6"
}

response = requests.post(url, json=payload)
print(response.json())
```
</RequestExample>

<ResponseExample>
```json Success Response
{
  "mensaje": "Guardado",
  "exito": true
}
```
</ResponseExample>

## Get All Exam 1 Submissions

<RequestExample>
```bash cURL
curl https://proyecto-ingenieria-software-6ccv.onrender.com/respuestas_examen1
```

```javascript JavaScript
const response = await fetch('https://proyecto-ingenieria-software-6ccv.onrender.com/respuestas_examen1');
const submissions = await response.json();
console.log(submissions);
```

```python Python
import requests

response = requests.get("https://proyecto-ingenieria-software-6ccv.onrender.com/respuestas_examen1")
submissions = response.json()
print(submissions)
```
